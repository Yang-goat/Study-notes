# day-5复习拓展

## 复习

```
web前端 -- 写一个网页给用户看的内容
为什么要学前端 -- 以后的实战项目会涉及到独立开发一个网站/小程序
了解前端是怎么设计的(我们这个阶段只是对前端进行补充 并不会很深入的学习)


前端三剑客:
    html:超文本标记语言(不是编程语言)
    超文本:比普通文本更强大更厉害,可以带图片,表格表单,音乐,视频,超链接
    标记语言:里面的代码是以标签的形式书写,在网页显示,类似于一块块拼图
    css:层叠样式表
    美化网页(排版 颜色 大小 性能优化)
    让网页看起来更好看 性能更好
    javaScript:编程语言/脚本语言
    主要做动态样式/前后端网页交互
    
身体-->装饰-->蹦迪(行为)

HTML里的代码称为标签:
1.单标签:一般用来实现特定功能<标签名>
	<hr>分割线
	<br>换行
	<img>图片
2.双标签:一般是用来包裹内容会有一个闭合符号/ 写在结束标签名<标签名></标签名>
	a,b,p,span,strong,h1-h6,div
	<a></a>
	
网页基本框架:
<html></html>:网页的核心,网页内容都在里面
<head></head>:头标签,里面写的内容,主要是给浏览器看的 网页介绍,样式,属性设置
<body></body>:身体标签,给用户看的内容都在里面

写在head标签里的内容:
    <meta>:元标签,主要写网页说明信息,设置信息 方便搜索引擎来搜索到你这个页面信息
    <title>:标题标签 标题内容都是放在这里
    <style>:标签定于元素css样式
    <link>:链接标签,用来链接外部文件(css)

写在body标签里的内容:
1.文本标签:
    <h1>-<h6> 1-6级的标题标签(一般一个网页只有一个h1标签)
    <p> 段落标签 自带换行
    <span> 文本标签 没有语义性 不会换行 一般结合css加样式
    <b>-<strong>粗体标签
    <i>-<em>斜体标签
    <u>-<ins>下划线标签
    (后者带有强调的作用 语义化)
 
2.功能标签
    <hr>分割线
    <br>换行
    
3.转义字符
    &lt;小于符号
    &gt;大于符号
    &nbsp;空格
    
4.图片标签
	img的四要素
	src:图片路径
	alt:图片描述
	width:图片宽(可以不用单位)
	height:图片高(可以不用单位)
	<img src="图片路径" alt="图片描述" width="图片宽" height="图片高" title='图片标题'>
	写项目的时候尽量用相对/网络路径
	
注意:如果要用pycharm打开html的话最好使用相对路径
pycharm会将打开的文件夹作为项目目录
因为运行之后浏览器会更改位置

5.超链接a标签:
    作用:可以指定网页跳转指定的位置
    1.跳转到一个指定的网站
    2.跳转到网页里的一个指定的位置
    3.跳转到一个指定的文件
    
    语法:
    <a href="跳转位置">描述内容</a>
    锚点:
    使用a标签设置href属性名为#id名字 点击a标签就可以跳转所需位置
    <p id="nice">等我</p>
    <a href="#nice">点我</a>
    a标签打开网页的方式 可以通过target值修改
    _blank 新网页
    _self 原网页(一般不修改 默认)
    
6.列表标签:
	在网页里面显示一些有关联性的数据
	有序/无序列表子元素都是li 内容是写在li里 不要写在ul/ol里
	
    ul>li --无序列表--默认样式为黑点(嵌套的话嵌套里是白点)
        <ul>
            <li></li>
        </ul>
    ol>li --有序列表--默认样式为数字.可以通过设置type属性修改
        <ol>
            <li></li>
            <li></li>
        </ol>
    dl>dt>dd --自定义列表.没有样式 但是dd带一个缩进
    dl=大容器
    dt=小容器
    dd=列表内容
        <dl>
        	<dt>
                <dd></dd>
                <dd></dd>
                <dd></dd>
            </dt>
        </dl>	    
注意:缩进

7.盒子标签:
	作用:本身没有任何含义,作为划分网页区域的布局 可塑性很强
	盒子默认没有宽高 颜色 刚开始是看不到效果 相当于裸奔状态 需要开发人员(给他穿衣服)设置样式才看得到
	width:宽度 常用单位px
	height:高度 常用单位px
	background-color:背景颜色
	margin:外边距
	padding:内边距
	设置盒子居中: margin 任意中 auto;
	快速创建:
	一个id为xx的盒子  #xxx
	一个class为xx的盒子  .xxx
	
css:
层叠样式表--> 优化网页 让网页更加美观
层叠:多个选择器可以同时作用于同一个标签 效果进行叠加
样式表:元素视觉表现的集合 样式是依附在html而存在

html是写一个网页 css是找到网页里的内容 加特效;
	1.怎么找
	2.加什么样式

选择器-->找到标签的方式
*:通配符选择器 选中所有的标签(清除内外边距)
标签选择器:选中指定的标签(一般是清除样式)
类名选择器:给标签设置class类名 在样式表通过.class名方式指定
id选择器:给标签设置id名 在样式表通过#id名方式指定

样式表的写法:
1.行内样式表:把样式写在标签里 权重最高
(1.结构样式没有分离 不利于后续的维护 2.样式造成资源浪费 代码冗余)
2.内部样式表:把样式写到head标签里的style标签中
3.外部样式表:把样式写在专门的.css文件中 再通过link标签引入

背景图片和img标签使用区别:
如果为了给盒子加底色为内容部分加背景 或者是加一些小图标项目符号等 则使用背景图片.如果是确却的想表达一个图片的意思 则使用img标签
重要就使用img 不重要背景图

边框样式:
border:1px solid #096
border-weight:边框粗细
border-style:边框样式
border-color:边框颜色
border-radius:边框圆角

边框四个方向的值:
top-上 right-右 left-左 bottom-下
比如:margin-left

标签属性:
	1.块元素(block):独占一行有独立的宽高地盘 默认宽度为100% 高度为内容大小 div p ul li 
	2.行内元素(inline):没有独立的地盘 宽高就是自身大小 行内元素都是挤在一起 span b img a 
	3.行内快元素(inline-block):兼具了两者的优点特点 有空格尾巴
	
	display:类型转换
	display:inline-block
```

    <meta charset="UTF-8">

        #album{
            margin:100px auto;/*距离上方100像素 左右自动居中*/
            padding: 20px ;/*内边距*/
            text-align: center;/*内容居中*/
            width: 600px;
            height: 580px;
            background-color: red;
        }
        #album img{/*选中album里所有的图片标签*/
            width: 280px;
            height: 190px;
        }
        /*伪类选择器 通过样式的方式给标签添加属性效果 当鼠标经过Img图片就触发里的代码*/
        #album img:hover{
            transform: scale(1.5); /*transform变形标签 scale缩放*/
            transition:1s;/*延时操作 设置变形完成时间 加个过度*/
        }
    </style>
</head>
